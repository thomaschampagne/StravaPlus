About Elevate and Github
Elevate code source is hosted on GitHub (a social open-source development platform). This implies that Elevate's development workflow including features, bugs, and open-source contributions are managed in the Elevate project repository.
Before reporting a bug please check if someone else has already reported your problem or request:
  • First check the Frequently Asked Questions
  • Then check for existing issues or requests on GitHub
Your problem or request is unknown. How to report it:
Using a GitHub account submit your bug or request here.
You may don't want to create a github account with your primary email?
This is understandable. Just create a "temporary email" with tempail.com and register to github.com with this email to report your request/bug.
